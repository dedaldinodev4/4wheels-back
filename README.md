# 4Wheels-Back

## Stacks
* Nodejs
* Express
* Prisma
* Typescript
* MySQL
* SOLID Principles

## Clone & Open
```
    git clone https://github.com/dedaldinodev4/4wheels-back.git
    cd 4wheels-back
```
## Install & Run
```
  yarn install or npm install or pnpm install
  yarn dev or npm run dev or pnpm run dev
```
